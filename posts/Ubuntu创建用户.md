# ubuntu中增加用户 

```bash
sudo useradd -m [username] -s /bin/bash #创建账户，使用/bin/bash作为shell
sudo passwd [username] #设置密码
sudo adduser [username] sudo #添加管理员权限
su [username]#切换用户
```
